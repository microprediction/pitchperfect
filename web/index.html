A soccer value function you can touch.

pitchperfect ports a trained neural-network value function for football — V(s), the probability-flavoured estimate of which team scores next — into a tiny numpy package and an interactive browser demo. Drag 22 players around a real pitch and watch the model reason about danger in real time.

Open the interactive demo → View on GitHub

How it works

One network, three runtimes. The model is trained in PyTorch in the value repo. We export its weights once; pitchperfect reimplements the forward pass in numpy (the pip package) and in JavaScript (this site). A parity test proves all three agree to ~1e-6.
Deep Sets + attention. SoccerNetV6 is permutation invariant within each team, uses cross-team attention so players "see" each other, and outputs a single signed value V = 2·σ(logit) − 1. It generalizes naturally beyond soccer to any spatial team sport.
Probe what it knows. The demo computes interpretable formation metrics live — defensive compactness, width, depth, block area, line height — so you can connect a shape change to a value change.
Marginal value surfaces. Pick any player and sweep them across the pitch; the heatmap shows V at every position with the others held fixed — the model's view of where that player is worth the most.

Install

pip install pitchperfect

from pitchperfect import ValueNet
net = ValueNet.load()
v = net.value(
    ball=[0, 0],
    blue=[[-46,0],[-30,-18], ...],   # 11 [x,y] in sim coords (x∈[-50,50], y∈[-30,30])
    red=[[46,0],[30,18], ...],
)
print(v)   # signed value in [-1, +1]; +1 favors blue
