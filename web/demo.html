What am I looking at?

The value function. A permutation-invariant Deep-Sets + cross-team-attention network (SoccerNetV6), trained to predict which team scores next. V = 2·σ(logit) − 1: +1 means blue is almost certain to score next, −1 means red is.
The probes. Interpretable formation descriptors — defensive compactness, width, depth, convex-hull area, line height — computed live so you can watch how reshaping a block moves the value, not just that it does.

The exact same network runs three ways — PyTorch (training), numpy (the pitchperfect package), and the JavaScript on this page — verified identical to ~1e-6 by the parity test. See the free-kick application for a related value-based study.
